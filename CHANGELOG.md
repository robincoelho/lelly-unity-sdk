# Changelog
Todos os registros de alterações notáveis para este pacote serão documentados neste arquivo.

## [1.1.0] - 2026-04-23
### Adicionado
- **Lelly AI Control Center**: Nova interface customizada para o Inspector.
- **Editor Tester**: Permite testar Chat e Geração de Conteúdo sem entrar no modo Play.
- **System Instructions**: Campo para configurar o prompt da IA diretamente no componente.
- **Suporte a Overrides**: API agora aceita instruções de sistema por sessão.

## [1.0.0] - 2026-04-23
### Adicionado
- Versão inicial do SDK UPM.
- Suporte a Chat, Sessões e Geração de Conteúdo.
- Singleton MonoBehaviour `LellyAPI`.
- Componente `LellyManager` para integração facilitada.
